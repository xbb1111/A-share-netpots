# 行业研究画布设计

## 目标

将行业页升级为可研究、可编辑、可复用的投资研究画布：用气泡云图观察全市场行业强弱，用层级树浏览行业分类，用无限层级产业链组织细分逻辑，并把股票、分支指数和价格纪律连接为连续工作流。

## 页面与交互

- **今日行情**使用紧凑气泡图：每个行业为一个不重叠圆形，半径可按热度或涨幅切换，红涨绿跌；点击后优先进入唯一映射的产业链节点，未映射或映射歧义时进入行业详情。
- **行业全景**使用可展开树：一级节点展开二级，二级节点展开三级；叶子节点加载成分公司。展开状态在当前 URL 中保留，页面刷新后恢复。
- **产业链主题**将热门主题置顶，并提供“我的产业链”入口。选中主题后显示上游、中游、下游及其任意深度的子分支；选中节点时显示节点统计、股票和下一级节点。
- 节点中的股票支持名称/代码搜索和一键加入；点击股票跳转工具箱并预填价格纪律工具。点击任意分支生成临时指数预览，支持等权和市值加权，确认后才保存为工具箱自定义指数。

## 自定义产业链与分享

- 自定义产业链采用本地存储，结构包含名称、说明、根节点、嵌套子节点和去重后的股票标的；每个节点都可新增、重命名、删除、排序、添加子节点和管理股票。
- 分享使用 URL hash 中的编码 JSON；“加载链接”弹窗粘贴链接后校验版本、结构、节点 ID 与股票代码，再导入为本地副本。超过 URL 安全长度时改为显示可复制的导入文本，不创建可能失效的链接。
- 保存覆盖当前本地版本；导入永远创建新副本，不覆盖现有产业链。

## 数据、统计与工具联动

- 分支统计基于该节点及所有后代节点的去重股票：显示平均涨跌幅、平均市值、平均 PE、有效 PE 公司数/总公司数。平均 PE 排除缺失值、零和负值；其他指标仅以有效数值计算并显示覆盖数。
- 市值加权预览以实时市值为权重；市值缺失时禁用市值加权并显示原因。等权预览始终可用。临时预览不写入自定义指数存储。
- 通过共享工具导航指令把行业页操作交给工具箱：`price` 指令包含股票代码，`index-preview` 指令包含分支股票与权重方式。工具箱消费指令后打开正确面板、加载数据并清除一次性指令。
- 继续使用同源/Worker 行情 API 获取行业、成分股、PE 和市值；任一数据缺失时保持结构可浏览，指标显示 `—` 和覆盖说明。

## 测试与验收

- 气泡布局：无重叠、指标切换、涨跌颜色、气泡点击映射和歧义回退。
- 树分类：一级到三级展开/收起、叶子公司加载、URL 恢复。
- 自定义树：任意深度编辑、股票搜索去重、存储、分享编码、链接导入和超长回退。
- 分支统计：后代去重、平均涨幅、市值加权、正 PE 过滤与覆盖数。
- 工具联动：个股进入价格纪律、分支临时预览、等权/市值加权切换、确认保存到自定义指数。
- 回归：全量测试、生产构建、GitHub Pages 上通过 Worker API 获取数据；桌面和窄屏无横向溢出。

## 默认与边界

- 首版支持 A 股和现有热门产业链；行业树只展示数据源可验证的层级关系，无法确认父节点的条目归入“未归类”而不虚构关系。
- 不增加账户、云端数据库或公开协作编辑；分享仅传递用户主动生成的内容。
- 分支临时预览采用日线、基准值 100、默认等权；保存后用户可在工具箱中进行自定义权重与其他配置。
